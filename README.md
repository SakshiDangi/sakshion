# Sakshion

An adaptive AI learning platform built as a modular monorepo.

## Tech Stack

- Next.js
- TypeScript
- pnpm Workspaces
- Turborepo

## Repository Structure

```text
apps/
packages/
docs/
scripts/
```

## Getting Started

Install dependencies:

```bash
pnpm install
```

Start development:

```bash
pnpm dev
```

Build:

```bash
pnpm build
```
